# 🔐 Password Generator

A modern GUI-based Password Generator developed using Python and Tkinter as part of my CodSoft Python Programming Internship.

## ✨ Features

- 🔢 Custom Password Length Selection
- 🔒 Strong Random Password Generation
- 📋 Copy Password to Clipboard
- 🎨 Black & Blue Modern Theme
- 🖥️ User-Friendly Graphical Interface
- ⚠️ Input Validation and Error Handling

## 🛠️ Technologies Used

- Python 3
- Tkinter (GUI Library)
- Random Module
- String Module

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/yallapuabhiram/CODSOFT.git
```

2. Navigate to the project folder:

```bash
cd Task3_Password_Generator
```

3. Run the application:

```bash

```

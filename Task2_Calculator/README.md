# 🧮 Smart Calculator

A modern GUI-based calculator developed using Python and Tkinter as part of my CodSoft Python Programming Internship.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📊 Modulus (%)
- ⌫ Backspace Button
- 🗑️ Clear (C) Button
- 🎨 Black & Blue Modern Theme
- 🖥️ User-Friendly Graphical Interface

## 🛠️ Technologies Used

- Python 3
- Tkinter (GUI Library)

## 📂 Project Structure

```text
Task2_Calculator/
├── calculator.py
└── README.md
```

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/yallapuabhiram/CODSOFT.git
```

2. Navigate to the project folder:

```bash
cd Task2_Calculator
```

3. Run the application:

```bash
python calculator.py
```
